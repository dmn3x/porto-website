# Dhafin — Portfolio Website

Personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## Tech Stack

- HTML5 / CSS3 / JavaScript
- [GSAP](https://greensock.com/gsap/) — animations
- [@studio-freight/lenis](https://github.com/studio-freight/lenis) — smooth scroll
- Google Fonts (Poppins) + ChunkParty (local)

## Project Structure

```
porto-website/
├── index.html          # Main entry point
├── style.css           # Global styles
├── assets/             # Images and icons
│   ├── logo.png
│   └── arrow.png
├── fonts/              # Local fonts
│   └── ChunkParty.ttf
├── docs/               # Reference files
│   ├── figma-reference.png
│   └── Screenshot_*.png
├── package.json
└── .gitignore
```

## Getting Started

```bash
npm install
```

Open `index.html` in browser — no build step needed.
